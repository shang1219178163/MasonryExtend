v1.0.3 · CocoaPods · SPM

Masonry 扩展，九宫格一行到位

在 Masonry 之上补齐九宫格分布与 Content Hugging / Compression 链式写法，少写约束样板，多写业务布局。

iOS 9+ tvOS 9+ ObjC · Masonry MIT
Install 
dependencies: [
    .package(
        url: "https://github.com/shang1219178163/MasonryExtend.git",
        from: "1.0.3"
    )
]
pod 'MasonryExtend'

模块速览

按场景浏览常用 API。输入关键词可即时筛选。

SK
九宫格

固定间距分布

Item 尺寸自适应，行间距 / 列间距固定，按 warpCount 折行排布。

[list mas_distributeSudokuViewsWithFixedLineSpacing:5 fixedInteritemSpacing:5 warpCount:3 inset:UIEdgeInsetsMake(10, 10, 10, 10)];
SZ
九宫格

固定尺寸分布

指定 Item 宽高，间距随容器自适应，适合按钮墙、图标矩阵。

[list mas_distributeSudokuViewsWithFixedItemWidth:120 fixedItemHeight:120 warpCount:3 inset:UIEdgeInsetsMake(10, 10, 10, 10)];
FX
九宫格

尺寸 + 间距全固定

可由内容撑开 SuperView；warpCount 大于数量时自动填充空白 View。

NSArray *filled = [list mas_distributeSudokuViewsWithFixedItemWidth:0 fixedItemHeight:0 fixedLineSpacing:8 fixedInteritemSpacing:8 warpCount:3 inset:UIEdgeInsetsZero];
HG
优先级

Content Hugging

在 mas_makeConstraints 内链式设置抗拉伸优先级，贴近 Masonry DSL。

[label mas_makeConstraints:^(MASConstraintMaker *make) { make.hugging.horizontal.priorityHigh(); make.hugging.vertical.priority(250); }];
CR
优先级

Compression Resistance

链式配置抗压缩，避免标签被挤压；支持 Required / High / Low 等档位。

[title mas_makeConstraints:^(MASConstraintMaker *make) { make.compression.horizontal.priorityRequired(); make.compressionResistance.vertical.priorityLow(); }];
UV
UIView

UIView 优先级属性

直接读写四轴优先级属性，不必每次走进 Constraints Block。

view.horizontalHuggingPriority = UILayoutPriorityDefaultHigh; view.verticalCompressionResistancePriority = UILayoutPriorityRequired;
